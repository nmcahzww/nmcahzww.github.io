聊聊最近 part2

2020-08-03
日常
1578 words 4 mins read times read

Contents

  • 工作

工作

终于有空闲下来更新一下了，距离上次的话主要是成功的换工作啦。虽然还是和以往的风格一样， 面了一家->通过->直接入职 ，但是也算是为 这段时间的准备添上了一个结果。 面字节真的很坎坷，我其实还算是一个挺认得清自己能力的人，一共面了三个组，前两个组都是到了终面把我拒了其实还是挺难受的，而且理由竟然是看不到学习的热情… 不是吧阿sir

我的缺点我是清楚的，技术和经验的积累有限，并且我确实遇到/见识过一些很强很强的人。但是以我的视角来看，我觉得我申请的职位职级，我一定是比绝大部分人要强的， 不管是技术能力上还是业务能力上。当然，我不是以此作为被拒的借口，只是从事实的角度上，单纯的对“背几天网上搜到的内容（e.g.某中间件）<=> 会用/有经验 <=> 可以胜任工作” 且诸多情况以此作为面试依据，感到有一点失望，能理解同时也很难认同。可能是我的 level 还太低:)

现在回忆起那么多轮面试，其实真的没有很多有价值的东西。坦白讲，很多我回答的内容，都是临时学的，而工作中真正有价值的思考和经验，nobody cares. 我觉得我能接受因为 coding 因为一些技术素养而拒我，或者因为我的技术思维我的价值观等等拒我，但是我很难理解会因为没用过 kafka 拒我，只是工具而已…

入职之后其实也没什么特殊的体验，就是有点加班吧，再一个就是接手了一个超级古老的代码，真的连看都不想看。感觉这边很多人都不在乎代码的质量和风格， 我其实是很纠结这个事情的，让我觉得很难受，我的上家公司就好很多，可能是因为小公司吧，大家有时间有精力有能力去落实。 即使是做技术难度很有限的纯业务 coding，漂亮、设计优秀的代码依然是基础， 但是感觉这边的思路重点上没有「漂亮」的代码这一环，永远是 堆功能->重构 的循环。我也比较懒，我觉得有问题我会提，至于改不改的我真的懒得催， 就是我自己的开发热情真的很受影响，不过我入职到现在也没写过几行代码:) whatever

好像一直在吐槽？但是还是碰到了很厉害的人例如我 mentor，让我对技术和业务的关系有重新的思考和审视。做的一些国际化的事情还是挺有意思的， 毕竟不是所有公司都有类似的场景，但是没做两天 tiktok 直接上新闻，很难受。

生活

入职的时候说让我去北京，team 大部分人都在北京，我不想去，就让我 base 深圳了，但是还是没逃过来北京出差，疫情突然就稳定了…

北方真的是不下雨啊，虽然我一直只在 酒店-公司，但是管中窥豹，还是能感受北方，干，萧瑟。深圳除了夏天太热 + 可能有蟑螂 + 城市基建太差， 其他都挺好的，主要是还挺靠近海。住酒店住的有点反胃了，五星级酒店床头只有一个插座是真的难受。 想着周末自己在北京无聊，出去逛逛，竟然也只去诊所看了看牙，还是打车去打车回，一点都没有在外面停留。team 在北京的话，工作上其实还是挺好的， 有种团队的感觉，但是生活上我觉得我撑不了多久了，得赶紧回去深圳。

吃的话，公司食堂还可以，没吃过别的 base 的食堂，但是深圳>北京，刚来的时候因为疫情只能吃盒饭，要哭了。北京的外卖，又贵又不好吃， 不知道是我住的位置不好还是吃不习惯…吃 kfc 之类的又怕胖，难受住。

想想也算是在好几个城市生活过了，苏州-西安-杭州-深圳-北京，确实挺能折腾的，也不知道还能折腾多久。身边开始陆续冒出 98、99 年的小朋友了， 还是能感受到自己长大了。最近偶尔会想起杭州，其实杭州真的还可以，生活方面真的挺舒服的，有缘再回去吧。

去深圳的时候买了胶囊咖啡机，能直接喝 espresso 的都是猛男吧…刚开始不太懂什么都没加直接干了一杯，喝完瞬间明白了自己的咖啡取向。而且因为买了不同口味的胶囊， 我以为是口味的问题，很苦很酸，所以换了一个上面标注酸度低的又试了一次。后来才知道这是用来打底的，心疼自己。

大家都说工作决定生活，有时候想想生活才决定工作啊，还是要努力丰富生活。
